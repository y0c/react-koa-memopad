# Memopad

A Simple Markdown memopad built with koa & react

This project was developed for personal learning purposes.

## Feature

* Social Login
* JWT based Authentication 
* Markdown posting

## Stack

### Server

* koa
* serverless-http
* mongoose

**deploy - aws lambda**

### Front

* react
* redux
* redux-pender
* styled-components
* semantic-ui-react
* hellojs

**deploy - aws s3 with cloudfront**
